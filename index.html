HTML

  • >Hyper-text-markup-language

CSS

  • >Adds color & effects to webpages

Javascript

  • >All logic for moving parts of a webpage

Node.js

  • >Node Package Manager: manages pre-made code imported to projects
  • >Creates a manifest file that lists all modules imported to a project
  • >Makes installing & sharing modules simple

Git

  • >A program for storing & sharing files between collaborators
  • >Tracks the history of edits to a project and allows them to be viewed independently

GitHub

  • >A website & app used for creating remote repositorys for collaborating programmers
  • >Contains QoL features and access to a large coding community

Tailwind

  • >Node module that allows CSS to be written directly into individual HTML elements
